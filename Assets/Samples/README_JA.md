## サンプルシーンの利用方法
サンプルシーンは以下の手順で利用できます。

1. [NOVAシェーダのリポジトリ](https://github.com/CyberAgentGameEntertainment/NovaShader)をクローンする
2. クローンしたプロジェクトをUnityで開く
3. [Assets/Samples](Assets/Samples)内のシーンを開く
4. 再生する
